# Week 2 - Data Management (M3S2)

## Objective
Consolidate understanding of advanced JavaScript data structures — objects, Sets, and Maps — to organize, store, and manipulate information efficiently.

## Concepts Reinforced
- Creating and manipulating objects
- Using Sets for data uniqueness
- Using Maps for associative data
- Loops (`for...in`, `for...of`) and iteration methods (`forEach`, `Object.keys()`, `Object.values()`, `Object.entries()`)
- Validation for correct data

## Files
| File | Description |
|------|-------------|
| `index.html` | Main page with Tailwind CSS and the shared TextTarea component |
| `gestion_datos.js` | Core logic: product objects, Set operations, Map operations, iteration methods, validation |
| `style.css` | Additional background styling |

## How to Run
Open `index.html` in any modern browser and click the **Run** button.

## Tasks Completed
1. Product objects with id, name, price
2. Set creation, add, has, delete, for...of iteration
3. Map with category-to-product associations
4. Iteration with for...in, for...of, forEach, Object.keys/values/entries
5. Data validation for complete product info
